Grid

A grid helps you layout your page in a 12 column system. The grid is defined by rows and columns. To add responsiveness to the layout you can use screen size specific column widths.
This can be done by adding the screen size: row__col--{size}--{xs,s,m,l,xl}
Change the width of your browser to see the difference.

Hide content

Content can be easily hidden based on screen width.

There are 4 different ways to hide content.
Always hide content: hide
Hide content on a certain screen width: hide--{xs,s,m,l,xl}
Hide content on and above a certain screen width: hide--{xs,s,m,l}-and-up
Hide content on and below a certain screen width: hide--{s,m,l,xl}-and-down
